Projects

The SOABridge project is developing a framework which can be used to develop a dynamic messaging framework which will replace your static ESB with a Dynamic ESB. Our framework will provide you with the abillity to scout the route from one system to another and have that path remembered and modified on the fly when the end points change.

~Breeze-Scala~

This is our Scala version of our framework. The Scala language has been a big help with it's ability to reduce some of the development needed to create the components of our framework.

~Breeze~

This is our Java version of our framework. This version of the framework is constructed with plain old java objects (POJOs).
